# Ejercicios_HTML
Mis primero ejercicios de HTML sencillos, para el módulo LMSG

## Descripción
En este repositorio de GitHub vamos a ir subiendo ejemplos de páginas web realizadas para el módulo LMSGI.
Lógicamente, las primeras páginas serán muy sencillas y, poco a poco, irán creciendo en complejidad.

## Entorno de Desarrollo Integrado
Para crear y editar las páginas utilizamos el IDE Ligero **Visual Studio Code**.
Para enviar las páginas al explorador, utilizamos la extensión **Live Server** que hemos instalado en VS Code.
Para *interpretar* las páginas, en principio, nos sirve cualquier explorador (por ejemplo, Chrome o Firefox).

[Página oficial de VS Code](https://code.visualstudio.com/)
